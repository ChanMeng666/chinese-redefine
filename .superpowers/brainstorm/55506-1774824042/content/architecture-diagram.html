用户流程
新用户注册流程：
  访问网站 → 注册页 → 填写邮箱/密码 → Resend 发验证邮件
  → 点击验证链接 → 邮箱已验证 → 可以使用生成器

社交登录流程：
  访问网站 → 点击 GitHub/Google 登录 → OAuth 授权
  → 自动创建账号（邮箱已验证）→ 可以使用生成器

卡片生成流程：
  输入词语 → 检查限流（5次/分钟/用户）→ 调用 OpenAI
  → 服务端生成 SVG → 存入 cards 表（默认公开）→ 返回结果
  → 用户可下载 SVG / 在个人中心切换隐藏

画廊浏览流程：
  访问 /gallery → SSR 查询公开卡片 → 瀑布流展示
  → 搜索词语 → 服务端过滤 → 返回匹配卡片
